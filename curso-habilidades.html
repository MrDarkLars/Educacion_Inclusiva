Curso Adaptado

Lección 1: Herramientas de Accesibilidad

  • Botón para Aumentar el tamaño del texto
  • Botón para Disminuir el tamaño del texto
  • Navegación usando solo el teclado (tecla Tab)
  • Compatible con lectores de pantalla

Lección 2: Navegación Básica

  • Usa la tecla Tab para moverte entre elementos
  • Presiona Enter para activar botones
  • Los encabezados ayudan a organizar la información

Lección 3: Actividad Práctica

Practica lo aprendido en la plataforma principal:

  • Prueba los botones de Aumentar y Disminuir Texto
  • Navega usando solo la tecla Tab
  • Escucha alguna sección con el lector de voz
